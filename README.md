# 🎯 Image Deblurring Using GAN

## 📌 Project Overview
This project focuses on restoring blurred images using Generative Adversarial Networks (GANs). The model learns to convert blurred images into high-quality sharp images by training a Generator and Discriminator network in an adversarial manner.

---

## 🚀 Features
- Converts blurred images into sharp images
- Uses Deep Learning (GAN architecture)
- Improves image clarity and reduces blur
- Works on paired dataset (blurred & sharp images)

---

## 🛠️ Technologies Used
- Python
- PyTorch / TensorFlow
- OpenCV
- NumPy
- Matplotlib

---

## ⚙️ How It Works
- **Input:** Blurred image  
- **Generator:** Produces deblurred (sharp) image  
- **Discriminator:** Classifies real vs generated images  
- **Output:** Enhanced sharp image
