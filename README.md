# Image Deblurring using GAN

## 📌 Project Description
This project focuses on restoring blurred images using Generative Adversarial Networks (GANs). The model learns to convert blurred images into sharp and clear images.

## 🚀 Technologies Used
- Python
- OpenCV
- TensorFlow / PyTorch
- NumPy

## ⚙️ How It Works
- Input: Blurred Image
- Generator: Produces a sharp image
- Discriminator: Checks if image is real or fake
- Output: Deblurred Image

## 📊 Results
- Improved image clarity
- Reduced noise and blur
